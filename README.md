# auto-inv [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]

## Installation

```sh
$ npm install --save auto-inv
```

## Usage

```js
const autoInv = require('auto-inv');

autoInv('Rainbow');
```
